# XDU Note
这是我在XiDian University学习时所做的笔记。

笔记意在记录我学习生涯时度过的难忘(痛苦)时光，每一个字在我看来，都是我冥思苦想后来之不易的产物。

虽然别人难以看懂笔记的内容，但是我依然想把它放在一个显眼的地方，好给我的同学们看看今天的我又在学什么有趣的东西。

>悲的是白纸一张磕磕碰碰，喜的是乘风破浪已过万重山，好一部悲壮的奋斗史！
