# 长度数据
```latex
\parindent 17pt
\smallskipamount 3pt plus 1pt minus 1pt
\medskipamount 6pt plus 2pt minus 2pt
\bigskipamount 12pt plus 4pt minus 4pt
\fill 0pt plus 1fill
\stretch{n} 0pt plus nfill
\unitlength 1pt
```

# 长度设置
## 一般命令
```latex
\addtolength{\parindent}{2pt} 为\parindent加上2pt
\newlength{\mylength} 初始值为0pt
\setlength{\parindent}{17pt}
\setlength{\mylength}{2pt plus 2pt minus 2pt}
```
## 字符串命令
```latex
\settodepth{\mylength}{abcdf} 赋值字符串的深度
\settoheight{\mtlength}{Fun} 赋值字符串的高度
\settowidth{\mtlength}{abcdf} 赋值字符串的长度
%需要calc
\settototalheight{\mtlength}{abcdf} 赋值字符串每一个字符的长度之和
```

# 产生长度命令
## 水平长度
```latex
%不定长度
\hspace{}
\hspace*{} 位于开头结尾时不会被系统删除
\hphantom{abcd} 高度为零，宽度为字符串的空白支柱
\hfill\hfil
%定长度
\thinspace 0.17em\neg 
\medspace 0.22em\neg
\thickspace 0.27em\neg
\enspace 0.5em
\quad 1em
\qquad 2em
```
## 垂直长度
```latex
% 不定长度
\vspace{}
\vspace*{}
\addvspace{} 只能用于\\和\par之后，忽略较小的长度
\vfill\vfil
% 定长度
\smallskip
\medskip
\bigskip
\vphantom{abcd}
```
