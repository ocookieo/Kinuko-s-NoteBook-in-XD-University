# 弹性消失

1. 弹性长度和其它数字相乘会变成0pt的刚性长度。
2. 部分弹性命令的特殊规则，如h

# fill的忽略

fill对hfil的抑制，hfill在行末被忽略（加上`\hfill~`），vfill在一页之首会被忽略。

# fill的衍生命令
```latex
\makebox[6cm]{\dotfill}
\makebox[6cm]{\hrulefill}
\makebox[6cm]{\downbracefill}
\makebox[6cm]{\upbracefill}
\makebox[6cm]{\leftarrowfill}
\makebox[6cm]{\rightarrowfill}
```
