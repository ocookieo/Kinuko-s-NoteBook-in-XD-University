# 列表
有enumerate环境和itemize环境，最多叠加三次，但是建议不要叠加到两次以上。

使用示范：
```latex
\begin{frame}
	\begin{block}{开普勒定律}
		\begin{enumerate}
			\item 行星绕太阳运动的轨迹是椭圆
			\item 行星和太阳连线之间的面积在相同时间内是相等的
		\end{enumerate}
	\end{block}
\end{frame}
```
